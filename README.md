# 贪吃蛇游戏

一个使用HTML、CSS和JavaScript实现的经典贪吃蛇游戏。

## 游戏特点

- 简洁美观的界面设计
- 键盘方向键控制
- 触摸屏支持（适用于移动设备）
- 分数系统
- 游戏难度随分数增加而提高
- 响应式设计，适配不同屏幕大小

## 如何游玩

1. 点击"开始游戏"按钮开始
2. 使用键盘方向键（↑ ↓ ← →）控制蛇的移动
3. 在移动设备上，可以通过滑动屏幕控制方向
4. 吃到红色食物可以增加分数
5. 避免撞到墙壁或自己的身体
6. 游戏难度会随着分数增加而提高

## 游戏控制

- **开始游戏**：开始新游戏
- **暂停**：暂停当前游戏
- **重新开始**：重置游戏并开始新游戏

## 技术栈

- HTML5
- CSS3
- JavaScript

## 在本地运行

只需将仓库克隆到本地，然后在浏览器中打开`index.html`文件即可。

```
git clone https://github.com/Zeyu7233/snake-game.git
cd snake-game
```

然后在浏览器中打开`index.html`文件。
